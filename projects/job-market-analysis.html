How It Works

This project combines Python data processing with Tableau visualization to transform raw job posting data into meaningful career insights.

The Data Pipeline

I built a comprehensive Python script that processes LinkedIn job postings through several stages:

Job Title Standardization

Mapped 30+ job title variations (e.g., "Senior Data Scientist", "Lead Data Scientist") to 4 standardized categories: Data Scientist, ML Engineer, AI Engineer, and Data Engineer.

Intelligent Skill Extraction

Developed regex-based algorithm to identify 150+ technical skills from job descriptions and skill sections, including Python, AWS, TensorFlow, NLP, and more.

Data Aggregation

Generated CSV files for top skills per role, hiring companies, geographic locations, and monthly posting trends, ready for Tableau visualization.

Interactive Dashboard

Built Tableau dashboard with dynamic filters, select any job role (or view all) to see top skills, companies, locations, and hiring trends update in real-time.
